## How to run this on your device

- Clone this repository
```
git clone https://github.com/kingdreamerr/Day20-Day21_snake_game.git
```
- cd into the repo
```
cd Day20-Day21_snake_game
```

- Paste the following in the terminal 
```
python3 main.py
```
